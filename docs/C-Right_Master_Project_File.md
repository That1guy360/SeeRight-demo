
# C-Right (SeeRight) — Complete Master Project File

------------------------------------------------------------
EXECUTIVE SUMMARY
------------------------------------------------------------

C-Right (SeeRight) is a screen-based vision accessibility platform that adjusts digital displays to better match individual vision needs. 
It is built with a dual-mode system:

1. Classic Mode — No AI, fully offline, privacy-first.
2. Smart Mode — Optional AI-assisted personalization.

The system is designed for consumers, businesses, VR environments, and privacy-sensitive institutions.

------------------------------------------------------------
CORE CONCEPT
------------------------------------------------------------

"Make screens adjust to your eyes — not the other way around."

C-Right applies display-level rendering adjustments to improve clarity, reduce eye strain, and enhance readability on phones, tablets, desktops, and VR headsets.

It does NOT:
- Replace glasses
- Diagnose medical conditions
- Act as a medical device (unless regulated later)
- Collect unnecessary personal data
